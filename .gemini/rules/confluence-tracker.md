# Rastreador de Cambios para Confluence (Público)

## Objetivo
Generar un manifiesto de archivos tocados para documentar tareas de Jira (`KAN-XX`).

Al terminar una tarea:
1. Haz una lista de los archivos modificados.
2. Genera un archivo en `docs/task_manifests/KAN-XX.md` con una tabla descriptiva de los cambios.
3. Este archivo servirá como base para copiar y pegar la documentación técnica en Confluence.
